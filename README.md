# React Native Lab — Expo App

A React Native mobile application built with Expo demonstrating component-based mobile UI development.

## Overview
Explores React Native and Expo fundamentals by building a cross-platform mobile application with custom components, navigation, and asset management.

## Features
- React Native component architecture
- Expo SDK integration and asset handling
- Custom component with AssetExample
- Cross-platform iOS and Android support
- Expo Snack compatible

## Technologies Used
- **Framework:** React Native
- **Platform:** Expo
- **Language:** JavaScript

## Project Structure
```
+-- App.js               # Root application component
+-- index.js             # Expo entry point
```

```bash
npx expo start
```
